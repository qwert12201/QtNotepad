### **Qt-Notepad**
It's a simple notepad on PyQt6 + QSS. Notepad supports formatting, search, text replacement, language switching and more...

## **Installation**
  1. Install python higher 3.0 version and install pip
  2. Install and extract files in any folder, for example in `C:\project`
  3. Open cmd
  4. Go to installation directory: `cd C:\project`
  5. Install packages: `pip install -r requirements.txt`
  6. Run main.py: `python main.py`
